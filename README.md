# DATA.GOV.HK (data-gov-hk)

DATA.GOV.HK is a national government open-data portal for Hong Kong running **CKAN**, exposing a standardized catalog API over ~218 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-gov-hk/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=data-gov-hk-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.9.11

## API
- **DATA.GOV.HK CKAN Action API** — base `https://data.gov.hk/en-data/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://data.gov.hk/en-data)
- Live endpoint verified: `https://data.gov.hk/en-data/api/3/action/status_show` · ~218 datasets
